# Academic CV

This repo contains Kim Fernandes's LaTeX academic CV, permissively licensed for others to adapt.

The CV content and formatting follow academic standards and emphasize clear information presentation and typography. The CV's TeX file is automatically compiled to PDF and published whenever a commit is pushed to the main branch, via this [workflow](.github/workflows/build_publish.yml). This keeps the publicly distributed PDF up-to-date while avoiding commit history pollution from frequent changes to a binary file.
